
/*****************************************本工程实现功能**********************************************/

本工程使用了基本定时器（tim6、tim7）实现定时1s操作，LED每1s翻转一次，OLED每一秒计数加1




/*******************************************注意事项*************************************************/

1.注意不要删除工程文件夹的pro.uvoptx文件（.uvoptx后缀），否则会造成延时函数在仿真时延时时间是实际延时时间的2/3
（程序在下入板子后延时时间是正确的）.
若需要更改主工程文件名（pro.uvprojx），需要同时更改pro.uvoptx，保证两文件名一致

2.在使用软件仿真时进不去TIM7_IRQHandler(void)函数，但能进入TIM_IRQHandler(void)，不知道怎么回事，硬件仿真时
  可以正常进入

/*****************************************一些知识点**************************************************/
1.stm32f103vet6基本定时器包括tim6、tim7

2.TIM_ClockDivision:TIM Time Base Init structure成员，设定定时器时钟CK_INT频率与数字滤波器采样时钟频率分频比
，数字滤波器可见参考手册定时器框图，基本定时器无此功能。滤波器的工作原理如下：以采样频率f对信号进行采样，如果n个
采样点信号相同，则将信号输出。如果不同，则输出上一时刻采样值或其他特定值，频率越高滤波越好，但延迟越高。

3.由于TIM_TimeBaseInit在最后回产生一个更新事件，而更新时间和更新中断同时发生，故在使能TIM会立即进入一次中断
  若不想程序开始时进入中断，只需在TIM_TimeBaseInit与TIM_ITConfig之间清楚更新事件标志位即可
